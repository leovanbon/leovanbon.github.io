## Or just read it out

We can continue static analysis, notice that
![](attachment/Pasted%20image%2020260415235622.png)

There is a `revalidate()` method in `JTextBasket`:

![](attachment/Pasted%20image%2020260416000149.png)

Note that `public class JTextBasket extends JComponent`, so it passes the score as the component's name, then in `revalidate()` it `getName()` to get that score `6942069` for deriving the flag. Here is a python code of what it does:

```

from Crypto.Cipher import AES

import base64

  

val = (6942069 * 10 + 1) ** 4

  

key = bytes.fromhex(str(val))

iv = bytes.fromhex(str(val)[::-1])

ct = base64.b64decode("6Ach6HiD0JmCc1L+RwxDRzhW3sC1kS6XydgSuWVFpxVXRU8EjfuMxIMoIzMwK/ii")

cipher = AES.new(key, AES.MODE_CBC, iv)

decrypted = cipher.decrypt(ct)

print(decrypted)

```
